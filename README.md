# 微信砖了个砖游戏机器人

* 本代码仅作学习研究使用，请勿用于任何商业用途!!!

## 使用方法
```
python3 -m venv venv
source venv/bin/activate
python3 main.py 1layer.png
```

## 逻辑流程
1. 在图片中识别矩形，找到游戏的真实边框
2. 识别游戏框中的棋子
3. 使用 ncc 在棋子间进行类型识别
4. 使用深度遍历模拟游戏规则寻找消除路线
5. 输出消除路线
